# 🎓 CDY2204 – Sistema de Inscripción de Cursos

> Proyecto de microservicios con **Spring Boot** para la asignatura **Desarrollo Cloud Native**  
> Semana 4 | Fundación Duoc UC

---

## 📋 Descripción

Plataforma educativa que permite gestionar el catálogo de cursos virtuales y las inscripciones de estudiantes mediante una API REST. El sistema incluye despliegue automático en la nube mediante un pipeline CI/CD completo que publica la imagen en **Docker Hub** y la despliega en una instancia **EC2 de AWS**, con persistencia en **Oracle Autonomous Database (Cloud)**.
